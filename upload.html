Sport Adatok Feltöltése

Ezen az oldalon feltöltheted a sportágakat és kategóriákat tartalmazó JSON fájlt az Appwrite adatbázisodba.

Használati útmutató

  1. Az Appwrite konfiguráció automatikusan betöltődik az appwrite.js fájlból
  2. Add meg a Database ID és Collection ID értékeket
  3. Válaszd ki a JSON fájlt vagy használd a mintaadatokat
  4. Kattints a "Feltöltés" gombra
